# Loggin

Level
	ALL		:	모든 로깅
	TRACE	:	세일
	DEBUG	:	디버깅
	INFO	:	강조 정보
	WARN	:	경고
	ERROR	:	오류
	FATAL	:	심각한 오류(서버다운?)
	OFF		:	로깅해제
	
출력 패턴 설정

	%m		- 로그 내용 출력
	%p		- 레벨 출력
	%r		- 어플이 시작 후 이벤트가 발생하는 시점까지의 경과시간(밀리세컨드1/1000 단위)
	%c		- package
	%c{n}	- n(숫자)만큼의 package를 가장 하단부터 역으로 출력
	%n		- 개행문자(줄바꿈)
	%d		- 이벤트 발생날짜 : 출력패턴은 {}안에 정의
				예 ] %d{yyyy/MM/dd}
	%C		- 호출자의 클래스이름
	%M		- method 이름을 출력
	%F		- 프로그램 파일 명 출력
	%l		- caller 정보(호출된 정보)
	%L		- caller의 라인 수
	%x		- 쓰레드 관련 
	%X		- 쓰레드 관련
	%t		- 쓰레드 이름
	%%		- %를 출력(\와 같은 기능)
